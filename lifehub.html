⚡ Slack Integration

Connect a Slack channel so anything you post there pops up here — pick a category with one tap and it's saved as an Idea.

Step 1 — Create Slack App
Go to api.slack.com/apps
Create New App → From Scratch. Enable Incoming Webhooks and create a webhook for your ideas channel (e.g. #ideas). Copy the Webhook URL and paste below.
Step 2 — Create /idea Slash Command
Slash Commands → /idea
In your app settings add a Slash Command. When hosted, set Request URL to your LifeHub URL. For local use, the Simulate button below lets you test the full flow instantly.
/idea Post reel about modified Civics https://instagram.com/p/xyz
Step 3 — Paste Webhook URL
Enables confirmation messages back to Slack
